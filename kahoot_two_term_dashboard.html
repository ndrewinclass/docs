Every topic with paired PRE/POST quizzes showed improvement. File Permissions: 35% → 47%. Block Devices: 29% → 24–38%. SSH: 31% → 37%. The pattern holds even when topics are revisited across different class sections.

47 quizzes over ~5 months means ~2–3 sessions per week. Peak was Oct–Nov 2025 with 27 quizzes in 2 months. This aligns with the established "retrieval practice" learning effect - frequent low-stakes testing boosts retention.
